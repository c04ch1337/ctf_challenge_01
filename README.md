# CTF_CHALLENGE_01: The Lost Temple

## Story
Deep in the jungles of Cyberia lies the Lost Temple, a place rumored to hold ancient secrets and treasures. However, the temple is protected by layers of security and traps. Your mission is to navigate through the temple, uncover hidden flags, and learn about web security vulnerabilities along the way.

## Objective
Find all 10 hidden flags by exploiting vulnerabilities in the temple's web application. Each flag corresponds to an OWASP Top 10 vulnerability.

## How to Play
1. Start the challenge by running `./start.sh`.
2. Access the web application at `http://localhost/dvwa`.
3. Use your skills to find and exploit vulnerabilities.
4. Submit the flags to complete the challenge.

## Hints
If you're stuck, check the `hints.md` file for guidance.

Good luck, adventurer!
